## 功能

- 常驻实时云 — 状态栏常驻天气、步数等信息流
- 全应用上岛 — 不限官方白名单，所有通知均可胶囊化
- 原生歌词胶囊 — 状态栏实时歌词，多音源适配
- 胶囊/卡片定制 — 背景(无/纯色/图片/动态)、描边、圆角、月牙、封面形状、宽度
- 动态背景 — GIF / 视频帧序列作为胶囊或卡片背景 (星尘权益)
- 超高帧录屏 — 突破系统帧率与码率限制
- 游戏帧率显示
- 浮窗回复
- 状态栏图标排序

## 环境要求

- Android 14+
- Root
- LSPosed 或兼容 Xposed 框架
- 作用域: system、com.android.systemui 等 (安装后自动声明 staticScope)

## 使用

1. 安装 APK，在 LSPosed 中启用模块
2. 确认作用域已勾选 (模块已声明 staticScope)
3. 重启设备
4. 打开 AstraFlow 进行配置
